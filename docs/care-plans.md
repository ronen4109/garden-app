# תכניות טיפול שנתיות — Seed Data

מסמך זה מגדיר את תכנית הטיפול לכל אחד מ-6 העצים. ישמש כ-seed לטבלת `care_plan_items`.

**אזור:** בת חפר, מרכז ישראל (אקלים ים-תיכוני, חורף גשום וקיץ חם ויבש).

**עקרונות:**
- מינימום פעולות, מקסימום אפקט.
- דשן עיקרי: גרגירי איטי-שחרור (Multicote / Osmocote) — חופן 50-80 גרם פעמיים בשנה.
- חיפוי (mulch) שכבת קש/שבבי עץ — מפחית השקיה ב-30%.
- ריסוס שמן קיץ פעם-פעמיים בשנה לפי הצורך.
